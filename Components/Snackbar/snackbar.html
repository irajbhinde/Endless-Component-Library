Snackbar
Snackbars provide brief messages about app processes at the bottom of the screen.
Baseline Snackbar
Baseline Snackbars have a fixed position & can be used to display a message on the left end on the screen. In our component library, we are using it as an error message. 
                
            <-- Using the 'baseline' class name -->
        
            <div class="snackbar baseline">
                <span class="snackbar_text-error">Can't send photo. Retry in 5 seconds.</span>
                <button class="snackbar-btn">
                 <b> RETRY </b> 
                </button>
                < class="color-white far fa-times-circle fa-lg"></i>
            </div>

Leading Snackbar
Leading Snackbars have a fixed position & can be used to display a message on the middle of the screen. In our component library, we are using it as an success message. 
                
            <-- Using the 'baseline' class name -->
        
            <div class="snackbar">
                <span class="snackbar_text-success">Item Uploaded Successfully</span>
                <button class="snackbar-btn">
                 <b> Undo </b> 
                </button>
                < class="color-white far fa-times-circle fa-lg"></i>
            </div>
