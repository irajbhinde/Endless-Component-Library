Badges
Badges are small circles, positioned either at top-right, top-left or bottom-right of the parent component. Badge can be used to display numbers, online / offline status, depending on where they are used.
Text with a Badge

      <-- Text with a badge -->
  
      <div class="badge_type" > <p>Text with a badge </p>
      <span class="badge_number type1"> 2 </span> />
  
      <-- Text with a large badge -->
  
      <div class="badge_type" > <p>Text with a large badge </p> 
      <span class="badge_number type2"> 2 </span> />
Badge on Icons
You can apply badges on icons as well. Just add the below script tag to get the font awesome fonts.
3
3
3 
          
      <-- Insert the following in your script to get font-awesome fonts  -->
  
       <script src="https://kit.fontawesome.com/fbbd0ff1df.js" crossorigin="anonymous" ></script> 
  
      <-- Badge on Icons 1 -->
  
      <div class="badge_type" > <i class="fas fa-home fa-3x"> </i>
      <span class="badge_number type3"> 3 </span> </div>
  
      <-- Text on Icons 2 -->
  
      <div class="badge_type" > <i class="fas fa-envelope fa-3x"> </i>
      <span class="badge_number type4"> 3 </span> </div>
Badge on Avatars
Badges on avatars can be used to show the current status of a user. For example we have three class variants : available | offline | busy. Below is the example of available class, to access other status, replace it with the desired status. 
          
            <div class="badge_type" > <img class="avatar lg" src="attach desired image"> 
            <span class="badge_number available"> 3 </span> </div>
