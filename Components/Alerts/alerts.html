Alert
Alert messages can be used to notify the user about something special: danger, success, information or warning.
Simple Alert
Simple alerts can be used using a simple header_content class followed by the color of your choice. We are using four class variants : danger | success | info | warning 
            
              <div class="header_component ">
              <div class="header_content danger">
              <p> <b>Danger!</b> This header indicates a dangerous or potentially negative action </p> 
              </div>  
              </div>

Danger! This header indicates a dangerous or potentially negative action

Success! This header indicates a success or potential positive action.

Info! Info! Indicates a neutral informative change or action.

Warning! Indicates a warning that might need attention.

Outline Alert
Outline alerts are created by using header_content class, followed by outline-variant i.e danger | success | info | warning 
          
            <div class="header_component ">
            <div class="header_content outline-danger">
            <p> <b>Danger!</b> This header indicates a dangerous or potentially negative action </p> 
            </div>  
            </div>

Danger! This header indicates a dangerous or potentially negative action
