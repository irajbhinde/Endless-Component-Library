Buttons
Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap.
Primary & Secondary Button
Primary & Secondary are the mostly used types of button. It is a good practice to match your primary button color with the logo color. While the secondary color could be a complementary of primary color. 
          
            <button class="btn primary-btn">Primary Button</button>
            <button class="btn secondary-btn">Secondary Button</button>
Link Button
Link buttons can be used to redirect user to a particular link by doing a button click 
          
            <button class="btn link">Primary Button</button>
Floating Action Button
A floating action button (FAB) represents the primary action of a screen. For example - a + icon while writing a mail, direction icon while using Google Maps,etc. 
          
            
            <button class="btn floating-plus"> <i class="fas fa-plus fa-lg"> </i> </button>
            <button class="btn floating-direction"> <i class="fas fa-directions fa-2x"> </i> </button>
            <button class="btn floatin-compass"> <i class="fas fa-compass fa-2x"> </i> </button>
